# 六爻纳甲技能

> "六爻通神，纳甲断卦。" —— 传统易学

六爻纳甲是易经占卜中最常用的方法之一，以铜钱或数字起卦，纳天干地支、六亲、世应、六神，进行自动化断卦分析。本技能提供完整的六爻排盘和断卦功能。

## ✨ 核心功能

- ✨ 铜钱起卦（模拟抛掷三枚铜钱）
- ✨ 数字起卦（用户输入 6 个数字）
- ✨ 时间起卦（当前时间自动起卦）
- ✨ 纳甲装卦（天干地支纳甲）
- ✨ 六亲排布（父母、兄弟、子孙、妻财、官鬼）
- ✨ 世应定位（世爻、应爻）
- ✨ 六神排布（青龙、朱雀、勾陈、螣蛇、白虎、玄武）
- ✨ 五行生克分析
- ✨ 自动化断卦（吉凶判断、断语）
- ✨ 应期推算（应验时间）
- ✨ 特殊格局识别（六冲、六合、伏吟、反吟）
- ✨ 纯 Python 实现，无外部依赖

## 🔧 技术特点

- 🔧 多种起卦方式（铜钱/数字/时间）
- 🔧 完整纳甲算法
- 🔧 自动化断卦逻辑
- 🔧 纯 Python 实现，无外部依赖

## 📖 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/liuyao-najia-skill.git
cd liuyao-najia-skill

# 确保 Python 3.8+ 已安装
python3 --version

# 赋予执行权限
chmod +x scripts/liuyao_pan.py
```

### 基本使用

```bash
# 铜钱起卦（模拟抛掷）
python3 scripts/liuyao_pan.py --method coin

# 数字起卦（输入 6 个数字，1-9）
python3 scripts/liuyao_pan.py --method number --numbers "3,5,7,2,8,9"

# 时间起卦（当前时间）
python3 scripts/liuyao_pan.py --method time

# 指定问事类型（带断卦分析）
python3 scripts/liuyao_pan.py --method coin --question 财运

# 输出 JSON 格式
python3 scripts/liuyao_pan.py --method coin --json
```

### 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --method | -m | 起卦方法 (coin/number/time) | coin |
| --numbers | -n | 6 个数字 (1-9，逗号分隔) | 可选 |
| --question | -q | 问事类型 | 通用 |
| --json | -j | 输出 JSON 格式 | 否 |

### 问事类型

| 类型 | 说明 |
|------|------|
| 通用 | 综合判断 |
| 财运 | 求财、投资、生意 |
| 事业 | 工作、升职、创业 |
| 婚姻 | 恋爱、婚姻、感情 |
| 健康 | 疾病、健康、养生 |
| 官司 | 诉讼、纠纷、是非 |
| 出行 | 外出、旅行、迁移 |
| 求子 | 怀孕、生育、子女 |
| 失物 | 寻物、找人 |
| 考试 | 考试、面试、竞争 |
| 交易 | 买卖、合作、签约 |

## 📚 六爻基础知识

### 六亲关系

| 六亲 | 关系 | 含义 |
|------|------|------|
| 父母 | 生我者 | 长辈、文书、房产、学业 |
| 兄弟 | 同我者 | 兄弟姐妹、朋友、竞争、破财 |
| 子孙 | 我生者 | 子女、晚辈、财源、快乐 |
| 妻财 | 我克者 | 妻子、财运、物质、下属 |
| 官鬼 | 克我者 | 丈夫、事业、压力、疾病、官司 |

### 六神特性

| 六神 | 特性 | 吉凶 |
|------|------|------|
| 青龙 | 喜庆、婚姻、贵人 | 吉 |
| 朱雀 | 口舌、文书、信息 | 中 |
| 勾陈 | 稳定、拖延、房产 | 中 |
| 螣蛇 | 变化、虚惊、怪异 | 凶 |
| 白虎 | 血光、疾病、凶灾 | 凶 |
| 玄武 | 欺骗、盗贼、暗昧 | 凶 |

### 特殊格局

| 格局 | 特征 | 含义 |
|------|------|------|
| 六冲卦 | 六爻相冲 | 事散、快速、不持久 |
| 六合卦 | 六爻相合 | 事成、缓慢、持久 |
| 伏吟 | 本卦变卦相同 | 呻吟、痛苦、停滞 |
| 反吟 | 本卦变卦相冲 | 反复、变动、不安 |
| 游魂卦 | 游魂归魂 | 心神不定、漂泊 |
| 归魂卦 | 游魂归魂 | 回归、安定、结束 |

### 用神选取规则

| 问事类型 | 用神 |
|----------|------|
| 财运 | 妻财爻 |
| 事业 | 官鬼爻 |
| 婚姻 | 男取妻财，女取官鬼 |
| 健康 | 官鬼爻（病）/ 子孙爻（药） |
| 官司 | 官鬼爻 |
| 出行 | 子孙爻（平安） |
| 求子 | 子孙爻 |
| 失物 | 妻财爻 |
| 考试 | 父母爻 |
| 交易 | 妻财爻 |
