# 六爻纳甲排盘技能 v3.0.0

> "京房易者，纳甲之宗，六爻之祖也。" —— 天工长老

## 1. Description

六爻纳甲是汉代京房所创的占卜方法，以铜钱起卦，纳干支、六亲、六神，断吉凶。

**v3.0.0 升级内容：**
- ✅ 完整用神旺衰计算（月令/日辰）
- ✅ 世应关系深度分析（生克关系）
- ✅ 六神详细断语
- ✅ 吉凶量化评分（0-100）
- ✅ 趋吉避凶建议生成
- ✅ 动爻详细分析

**v2.0.0 升级内容：**
- 完整纳甲装卦（天干地支）
- 四柱八字显示（年月日时干支）
- 六亲排布优化（精准五行生克）
- 六神完整排布
- 动爻、变卦计算

**核心功能：**
| 功能 | 说明 |
|------|------|
| 起卦 | 铜钱起卦 / 数字起卦 / 时间起卦 |
| 纳甲 | 每爻配天干地支 |
| 装卦 | 卦宫、世应、五行 |
| 六亲 | 父母、兄弟、子孙、妻财、官鬼 |
| 六神 | 青龙、朱雀、勾陈、螣蛇、白虎、玄武 |
| 动变 | 动爻标记、变卦推算 |
| 断卦 | 用神选取、旺衰分析、吉凶判断 |

## 3. How to use

### 调用方式

```bash
# 时间起卦（默认当前时间）
python3 ~/.openclaw/skills/liuyao-najia-skill/scripts/liuyao_pan.py

# 指定时间起卦
python3 ~/.openclaw/skills/liuyao-najia-skill/scripts/liuyao_pan.py --date "2026-03-21 15:00"

# 数字起卦（三个数字）
python3 ~/.openclaw/skills/liuyao-najia-skill/scripts/liuyao_pan.py --numbers "123,456,789"

# 铜钱起卦（6 次投掷，每次背面数 1-3）
# 1=少阳，2=少阴，3=老阳（动），4=老阴（动）
python3 ~/.openclaw/skills/liuyao-najia-skill/scripts/liuyao_pan.py --coins "3,2,3,2,3,2"

# 问事断卦（指定问题类型）
python3 ~/.openclaw/skills/liuyao-najia-skill/scripts/liuyao_pan.py --date "2026-03-21 15:00" --question "财运"

# 输出 JSON 格式（便于程序处理）
python3 ~/.openclaw/skills/liuyao-najia-skill/scripts/liuyao_pan.py --date "2026-03-21 15:00" --json
```

### 问事类型支持

| 类型 | 用神 | 说明 |
|------|------|------|
| 财运/交易/投资 | 妻财 | 财富、物质 |
| 事业/工作/官司 | 官鬼 | 职位、压力 |
| 婚姻/感情 | 妻财（男）/官鬼（女） | 配偶、恋情 |
| 健康/疾病 | 子孙/官鬼 | 医药、病症 |
| 考试/学业 | 父母 | 文书、成绩 |
| 失物 | 妻财 | 丢失物品 |
| 出行/旅行 | 子孙 | 平安、顺利 |
| 求子/怀孕 | 子孙 | 子女运 |
| 搬家/房屋 | 父母 | 房产、住所 |

## 6. 断卦规则

### 用神选取原则

1. **自占自事**：以世爻为用
2. **代占他人**：
   - 占父母：以父母爻为用
   - 占子女：以子孙爻为用
   - 占妻子/财物：以妻财爻为用
   - 占丈夫/工作：以官鬼爻为用
   - 占兄弟姐妹：以兄弟爻为用

### 吉凶判断步骤

1. **用神旺衰**
   - 得月令生扶为旺
   - 得日辰生扶为相
   - 休囚死为衰

2. **世应关系**
   - 世应相生：吉，事易成
   - 世应相克：凶，事多阻
   - 世应比和：平，需努力

3. **动爻分析**
   - 用神动化吉：吉上加吉
   - 用神动化凶：吉中藏凶
   - 忌神动：事有阻碍

4. **六神辅助**
   - 青龙临用：喜庆之事
   - 白虎临用：血光之灾
   - 玄武临用：防盗窃欺骗

### 特殊格局

| 格局 | 说明 | 吉凶 |
|------|------|------|
| 用神持世 | 用神在世爻 | 吉，事易成 |
| 用神空亡 | 用神逢空 | 凶，事难成 |
| 用神入墓 | 用神入墓库 | 凶，事被困 |
| 用神化退 | 用神化退神 | 凶，事渐衰 |
| 用神化进 | 用神化进神 | 吉，事渐成 |
| 六冲卦 | 卦逢六冲 | 事散、快 |
| 六合卦 | 卦逢六合 | 事成、慢 |
